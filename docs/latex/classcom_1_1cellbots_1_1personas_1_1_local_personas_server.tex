\subsection{Detailed Description}
Content Provider for serving personas. Persona files must be stored under\-: /sdcard/cellbot/personas/\-P\-E\-R\-S\-O\-N\-A\-\_\-\-N\-A\-M\-E/ Access persona files by using this pattern\-: content\-://com.cellbot.\-localpersonas/\-P\-E\-R\-S\-O\-N\-A\-\_\-\-N\-A\-M\-E/\-F\-I\-L\-E For example\-: /sdcard/cellbot/personas/default/happy.png can be accessed in the Web\-View by using\-: content\-://com.cellbot.\-localpersonas/default/happy.png I\-M\-P\-O\-R\-T\-A\-N\-T N\-O\-T\-E\-: For security reasons, you are not allowed to access anything with \char`\"{}..\char`\"{} or \char`\"{}//\char`\"{} in the path. This is to prevent a personas script from accessing the rest of the S\-D card.
